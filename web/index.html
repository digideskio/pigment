Pigment

Programming Language, Designed

goals

Pigment is an experiment in programming language design. We're taking modern programming language research and putting it in a thoroughly designed, interactive environment in the browser.

follow

Participate in development on Github or follow @dino_joel for updates.

Follow @dino_joel

research

Pigment is built on the work of many mathematicians and computer scientists. Notably - the Epigram team, who built the original implementation; the dependent types community, who continue to hammer out the theory; and the larger functional programming community.
